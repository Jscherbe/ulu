    base

    $base: 16px !default;

    Description

    Assumed pixel base, can change based on users font settings so this is just o get us in the ballpark) Note this is not the base font size but the user agent's or user's browser preferernce This number is just being used for calculating estimated em sizes from average base Since pixels are easier to understand but since we allow the user to set their font size All of our css is relative to that, including most of the layout (rems, other relative units)

    Type

    Number

    Used by

    • [mixin] set-base
    • [function] ratio-scale-size
    • [function] pixel-to-em
